# UniChat 应用程序用户手册

## 一、简介
UniChat 是一款智能助手应用程序，它结合了多种大语言模型（LLMs）和知识库文档，为用户提供准确且有用的答案。本手册将帮助你了解如何安装、配置和使用该应用程序。

## 二、安装

### 2.1 安装包获取
UniChat 以安装包形式分发。安装包构建完成后，会生成一个名为 `dist\unichat` 的目录，你可以将此目录复制到需要安装应用的设备上。

### 2.2 启动应用
将 `dist\unichat` 目录复制到目标设备后，只需双击该目录下的 `UniChat.exe` 文件，即可启动应用程序。启动后，会弹出一个自定义控制台窗口，同时应用会在后台启动 HTTP 服务器。你可以在浏览器中输入以下 URL 访问聊天界面：
```plaintext
http://localhost:63342/unichat/frontend/index.html
```

## 三、配置

### 3.1 环境变量配置
`.env` 文件用于存储环境变量，如不同大语言模型（LLM）提供商的 API 密钥和基础 URL。以下是一些常见的 LLM 提供商及其配置示例：
```plaintext
OPENAI_API_KEY=
MOONSHOT_API_KEY=
BAICHUAN_API_KEY=
ZHIPUAI_API_KEY=
DEEPSEEK_API_KEY=
DASHSCOPE_API_KEY=
```
你需要根据自己的需求，将相应的 API 密钥填入对应的字段。如果你没有相关的 API 密钥，可以联系相关提供商进行申请。
注意：你也可以直接把你的API密钥配置到环境变量，这样的话你完全不需要.env文件。

### 3.2 知识库配置
这一步在客户端页面上完成。
### 3.3 部署配置
你可以根据自己的需求选择合适的 LLM 提供商和模型，以及嵌入向量提供商和模型。如果 `LLM_MODEL` 或 `EMB_MODEL` 字段为空，应用程序将使用提供商部分定义的默认模型。

## 四、使用方法

### 4.1 启动聊天界面
启动应用程序后，在浏览器中输入 `http://localhost:63342/unichat/frontend/index.html`，即可打开聊天界面。

### 4.2 提问与获取答案
在聊天界面的输入框中输入你的问题，然后点击“发送”按钮，应用程序会将问题发送给智能助手，并在聊天窗口中显示答案。

### 4.3 配置模型和知识库
点击聊天界面右上角的配置按钮（齿轮图标），可以打开配置窗口。在配置窗口中，你可以选择不同的 LLM 提供商和模型，以及嵌入向量提供商和模型。同时，你还可以查看和管理知识库文档。

### 4.4 上传知识库文档
如果你需要上传自己的知识库文档，可以使用应用程序提供的 API。具体操作方法请参考开发文档或联系技术支持人员。

## 五、常见问题解答

### 5.1 模型未找到
如果应用程序提示某个模型未在本地找到，你可以使用 `Ollama pull <model_name>` 命令在终端中下载该模型。

### 5.2 网络错误
如果遇到网络错误，请检查你的网络连接，并确保 `.env` 文件中的 API 密钥和基础 URL 正确。

### 5.3 服务器错误
如果收到服务器错误，请查看应用程序的日志文件（`run.log`，位于应用程序根目录下）以获取更多详细信息。你也可以尝试重启应用程序。
